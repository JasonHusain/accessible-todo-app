How to Use This app


    This app is best used with a screen reader.


    App Layout

    This app consists of the following components:

    • Main controls, including the HELP button, main input and buttons to add tasks and clear the list
    • The current number of items in the list
    • The list, which can be scrolled vertically

    Adding Items/Clearing List

    • Enter a task in the main input
    • Use the "ADD TASK" button or press the "Enter" key to add an item to the list
    • Use the "CLEAR ALL" button to remove all items from the list

    Managing Individual Items

    Non-Editing Mode Controls

    • Use the "COMPLETE" button to toggle the completed status of a task. When marked "Complete", the text and background color will change. The Button will read "Not Complete"
    • Use the EDIT button to enable editing. all other items will be hidden.
    • Use the DELETE button to remove an item

    Editing Mode Controls

    • The task is now editable. Use the "SAVE" button or press "Enter" to save changes and disable Editing mode.
    • If the text field is empty, the original text will be restored upon saving.
    • Press the CANCEL button or the "Escape" key to discard changes and disable editing
