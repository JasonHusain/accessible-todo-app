HOW TO USE THIS APP

      Welcome to EasyTodos, a simple and accessible task management app.

      This app is designed to provide simple, accessible task management with minimal complexity. It is optimized for screen readers, keyboard navigation, and accessible touch interactions on mobile devices.

      To close this dialog, use the CLOSE button or press the "Escape" key.

      App List Views

      This app consists of two lists:

      • Active List
      • Completed List

      Use the "SHOW ACTIVE" and "SHOW COMPLETED" buttons to switch between lists.

      Active List Actions

      • Adding todos
      • Marking todos as Completed
      • Editing todos
      • Deleting individual todos
      • Clearing all active todos

      Completed List Actions

      • Marking todos as Active
      • Deleting individual todos
      • Clearing all completed todos

      Keyboard Navigation

      • Tab: Move to the next interactive element
      • Shift + Tab: Move to the previous interactive element
      • Alt + M: Go to main controls
      • Alt + I: Go to list items heading
      • Alt + L: Go to the first item in the current list
      • Alt + N or Alt + 1: Go to "SHOW ACTIVE" button
      • Alt + 2: Go to "SHOW COMPLETED" button
      • Alt + H: Go to "HELP" button

      Keyboard Actions

      • Enter: Add a todo to the active list or save edits
      • Escape: Discard edits or close help dialog
